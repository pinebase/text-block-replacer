Text Block Replacer



https://github.com/pinebase/text-block-replacer

# Author: Scott Forsberg
# Contact: contact.scottforsberg@gmail.com

# Release Date: 2019-06-10
# Version: .1



Description

	Replaces words found in an input file, and sends the converted text to standard output

	Example 1:
	
	$ ./text_block_replacer.pl cat blue in.txt
	one
	two
	three
	blue
	dog


	Example 2:
	
	$ ./text_block_replacer.pl cat blue in.txt > out.txt

	$ cat ./out.txt
	one
	two
	three
	blue
	dog



Prerequisites

	Perl



Installation

	Download and unzip the program files, then change working directory to them:
	
		# cd text-block-replacer


	Execute the program:

		# ./text_block_replacer.pl
